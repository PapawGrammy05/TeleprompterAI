# TeleprompterAI

## Features
- Multi-language support
- Speech recognition and voice commands
- Customizable UI
- Cloud sync for scripts

## Architecture
The application is built on a modular architecture, allowing easy integration of new features and enhancements. Key components include:
- Frontend: React.js
- Backend: Node.js with Express
- Database: MongoDB

## Tech Stack
- React.js for the frontend
- Node.js for the backend
- MongoDB for the database
- Docker for containerization

## How It Works
TeleprompterAI utilizes speech recognition technology to scroll scripts in real-time. Users can upload scripts, which are then processed and displayed on the screen.

## Setup & Installation
1. Clone the repository: `git clone https://github.com/PapawGrammy05/TeleprompterAI`
2. Navigate to the project directory: `cd TeleprompterAI`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

## Usage
- Upload your script via the upload button
- Use voice commands to control playback (start, stop, pause)
