# TeleprompterAI

An iPhone teleprompter app built in SwiftUI.

## Planned features
- Script editor
- Camera preview
- Speech-followed scrolling
- AirPods microphone support
- Vertical video layout for content creation

## Project structure
- App: app entry and main navigation
- Views: SwiftUI screens
- Managers: camera and speech logic

## Status
Early prototype in progress
